SaaSify - One Page SaaS Landing Template
========================================

ABOUT:
SaaSify is a modern, responsive one-page landing template for SaaS startups.

INSTALLATION:
1. Unzip the folder.
2. Open index.html in your browser to view locally.
3. Deploy via Netlify, Vercel, GitHub Pages, or your hosting provider.

STRUCTURE:
- index.html – main HTML file
- css/style.css – styling
- js/script.js – JS placeholder
- images/ – add your images here

CUSTOMIZATION:
Edit text inside index.html and swap images in the images/ folder.
